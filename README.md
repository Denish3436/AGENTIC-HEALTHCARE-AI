
# 🏥 Agentic Healthcare AI Platform

AI-powered healthcare platform with multi-agent systems, FHIR integration, and cloud-native DevOps architecture.

## 🚀 Quick Start

```bash
git clone https://github.com/denish3436/agentic-healthcare-ai.git
cd agentic-healthcare-ai

cp env.template .env
# update values

./deploy-azure.sh
````

---

💡 About the Project

This project demonstrates a **modern DevOps + AI architecture for healthcare systems**, combining:

* 🤖 Multi-Agent AI (Autogen + CrewAI)
* 🏥 FHIR-based healthcare data handling
* ☁️ Cloud-native deployment (Azure + Kubernetes)
* 🔐 Secure and scalable infrastructure

---

## 🏗️ Tech Stack

* **Cloud:** Azure (AKS, OpenAI, Key Vault)
* **Containers:** Docker + Kubernetes
* **CI/CD:** Azure DevOps Pipelines
* **Backend:** FastAPI / Python
* **Frontend:** React
* **AI Frameworks:** Autogen, CrewAI
* **Data:** FHIR APIs, PostgreSQL, Redis
